🎭 Nhân vật từ ảnh

Tải ảnh nhân vật (PNG nền trong)
vd: Pipo — dán lên tấm đứng trong cảnh
Ảnh nhân vật dán lên tấm luôn xoay về camera (billboard). Đặt vào cảnh 3D như vật thường — di chuyển, scale, nâng cao (giữ Alt khi kéo) để Pipo "bay" giữa các tòa nhà. AI sẽ thấy đúng nhân vật thật + đúng vị trí.

API Key (cho tính năng AI)

✕

Các tính năng AI dàn cảnh, dựng từ ảnh, tạo vật thể cần kết nối tới Claude API. Khi chạy offline (mở file trên máy), bạn cần dán API key của riêng mình vào đây.

Cách lấy API key
Vào console.anthropic.com, đăng ký / đăng nhập tài khoản (khác với tài khoản Claude.ai thường).
Vào mục Billing, thêm thẻ thanh toán & nạp một ít credit (API tính tiền theo lượng dùng — dựng vài cảnh rất rẻ).
Vào Settings → API Keys, bấm Create Key, đặt tên rồi copy ngay (key chỉ hiện một lần).
Quay lại đây, dán key vào ô trên rồi bấm Lưu. Key lưu trong trình duyệt máy bạn.

⚠ Key lưu trong trình duyệt máy bạn, không gửi đi đâu ngoài Anthropic. Đừng chia sẻ file đã lưu key.
ℹ Nếu mở file kiểu file:// (nhấp đúp) bằng Chrome, trình duyệt chặn lưu lâu dài — key vẫn dùng được trong phiên này, nhưng lần mở sau phải nhập lại. Muốn nhớ vĩnh viễn: dùng Firefox, hoặc chạy qua localhost (xem file hướng dẫn kèm theo).

✨ Storyboard cho AI Image

✕

Ảnh previz có gắn nhãn từng khối (Việt + Anh) giúp GPT-Image / Nano Banana hiểu mỗi khối là gì → ra hình đúng góc hơn. Tải ảnh này + copy prompt, dán vào AI tạo ảnh.

Depth/Edge map là tín hiệu MẠNH NHẤT để AI image hiểu góc máy. Đưa vào ControlNet (Depth hoặc Canny) trên Replicate/ComfyUI → góc giữ chính xác 100%, đổi style tự do. Mạnh hơn nhiều so với đưa ảnh khối thô.
Prompt mô tả (copy dán kèm ảnh)
Mẹo tăng độ chính xác
Khối thô khó nhận ra? Chọn khối → mục Texture 2D → dán ảnh tham chiếu thật lên khối (vd ảnh xe máy thật). Khối sẽ mang hình thật, AI hiểu ngay.
Đưa ảnh có nhãn cho GPT-Image/Nano Banana, gõ: "giữ nguyên bố cục và góc máy trong ảnh, chuyển thành [phong cách], các nhãn chỉ vật thể nào".
Muốn khóa góc tuyệt đối: dùng ảnh sạch làm input cho ControlNet (Depth/Canny) trên Replicate — góc giữ 100%.

Hướng dẫn & Phím tắt

✕
Quy trình cơ bản
Thêm chủ thể: bấm model ở cột trái (hoặc gõ ô tìm kiếm), hoặc dùng AI dàn cảnh / dựng từ ảnh bên phải.
Sắp đặt: chọn công cụ Di chuyển / Xoay / Tỉ lệ trên thanh giữa, rồi kéo vật thể trong cảnh.
Đặt máy quay: dùng chuột xoay quanh cảnh; chỉnh tiêu cự, khẩu độ, cao độ, góc máy nhanh ở cột phải.
Chụp shot: bấm nút Chụp shot — ảnh lưu vào danh sách, bấm để tải về làm tham chiếu storyboard.
✨ Storyboard: bấm nút Storyboard (cam) — xuất ảnh previz có nhãn từng khối (Việt+Anh) + prompt sẵn, để AI tạo ảnh (GPT-Image/Nano Banana) hiểu đúng góc & vật thể. Có nút "AI viết prompt điện ảnh" nếu đã nhập API key.
Lưu việc: tạo Project để app tự lưu offline; hoặc Xuất file .json để sao lưu.
