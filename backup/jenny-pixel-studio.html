API Key (cho tính năng AI)

✕

Các tính năng AI dàn cảnh, dựng từ ảnh, tạo vật thể cần kết nối tới Claude API. Khi chạy offline (mở file trên máy), bạn cần dán API key của riêng mình vào đây.

Cách lấy API key
Vào console.anthropic.com, đăng ký / đăng nhập tài khoản (khác với tài khoản Claude.ai thường).
Vào mục Billing, thêm thẻ thanh toán & nạp một ít credit (API tính tiền theo lượng dùng — dựng vài cảnh rất rẻ).
Vào Settings → API Keys, bấm Create Key, đặt tên rồi copy ngay (key chỉ hiện một lần).
Quay lại đây, dán key vào ô trên rồi bấm Lưu. Key lưu trong trình duyệt máy bạn.

⚠ Key lưu trong trình duyệt máy bạn, không gửi đi đâu ngoài Anthropic. Đừng chia sẻ file đã lưu key.
ℹ Nếu mở file kiểu file:// (nhấp đúp) bằng Chrome, trình duyệt chặn lưu lâu dài — key vẫn dùng được trong phiên này, nhưng lần mở sau phải nhập lại. Muốn nhớ vĩnh viễn: dùng Firefox, hoặc chạy qua localhost (xem file hướng dẫn kèm theo).

Quy trình cơ bản
Thêm chủ thể: bấm model ở cột trái (hoặc gõ ô tìm kiếm), hoặc dùng AI dàn cảnh / dựng từ ảnh bên phải.
Sắp đặt: chọn công cụ Di chuyển / Xoay / Tỉ lệ trên thanh giữa, rồi kéo vật thể trong cảnh.
Đặt máy quay: dùng chuột xoay quanh cảnh; chỉnh tiêu cự, khẩu độ, cao độ, góc máy nhanh ở cột phải.
Chụp shot: bấm nút Chụp shot — ảnh lưu vào danh sách, bấm để tải về làm tham chiếu storyboard.
Lưu việc: tạo Project để app tự lưu offline; hoặc Xuất file .json để sao lưu.
